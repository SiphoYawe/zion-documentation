Goal Check is Zion&apos;s system for tracking student progress through ACE PACEs: a daily goal is set each morning and actual completion is marked each evening. The Ritual is the full-screen, one-student-at-a-time way to run that same Goal Check data for an entire learning centre in a single guided session. Administrators, supervisors, and monitors can all run the ritual. This guide covers what is specific to the administrator role: seeing every centre, starting the ritual from your dashboard, and turning on optional goal approval.

<Note>
Administrators are not limited to an assigned learning centre the way supervisors and monitors are. Every active centre in your school shows up in your ritual centre selector, so you can start or resume the ritual for any of them.
</Note>

## The AM/PM ritual across centres

The ritual has two passes over the same roster, run at different times of day:

- **Morning Ritual (AM):** for each student, enter an end page for every active PACE. The start page is set for you automatically, based on the last page recorded.
- **End-of-Day Check (PM):** for each student, review the PACEs goaled that morning and record Hit, Missed, or Partial along with the actual pages completed.

Within either pass you can use Back to revise the previous student, Skip to move on and return later, the Roster sheet to jump straight to a specific student, and Skip remaining to close out the session early. Before a session finishes, it offers one more pass at any skipped students. Sessions save as you go, so closing the tab and coming back later resumes where you left off rather than starting over.

Because administrators are not tied to one centre, use the centre selector on the ritual card to check in on any centre&apos;s ritual, not only ones you supervise directly. Full step-by-step instructions for running the AM and PM passes live in the ritual reference page linked below.

## The Today&apos;s Ritual card on your dashboard

Your administrator dashboard includes a **Today&apos;s Ritual** card. If your school has more than one active learning centre, a **Learning centre** selector appears above the two ritual buttons so you can check any centre, not just ones assigned to you.

Each ritual mode shows one of three states:

| State | What you see | What happens if you tap it |
|---|---|---|
| Not started today | **Start Morning Ritual** / **Start End-of-Day Check** | Creates a new session for the selected centre and opens the full-screen ritual |
| In progress | **Resume Morning Ritual** / **Resume End-of-Day Check**, with an "X of Y handled" count | Reopens the existing session where it left off, no new session is created |
| Completed today | **Morning Ritual Done** / **End-of-Day Check Done** badge | Non-interactive; a secondary link takes you to the standalone Goal Check view (goal entry or progress marking) for that centre |

The card sits alongside, and never replaces, the standalone per-student Goal Check entry and progress-marking screens. Use whichever fits the moment: the ritual to move an entire centre through goal setting or end-of-day marking in one pass, or the standalone screens to adjust a single student.

## Optional goal approval

Zion supports an org-level setting that requires a sign-off on morning goals before they count as final. It is off by default, so goals entered in the ritual are live immediately unless you turn it on.

<Steps>
  <Step title="Turn on the setting">
    In School Settings, enable **Require goal approval in the Ritual**. When on, goals entered during the Morning Ritual start as pending until a supervisor or administrator signs them off.
  </Step>
  <Step title="Run or wait for a Morning Ritual session">
    Any completed AM session for a centre now carries goals in a pending state until approved.
  </Step>
  <Step title="Approve from the ritual summary">
    On the completion summary of a finished AM session, an approval panel shows a badge such as "3 goals pending approval." Tap **Approve all from this session** to sign off every pending goal from that session in one action. Once every goal is approved, the badge changes to "All goals approved."
  </Step>
</Steps>

Only an administrator or supervisor can approve goals. A monitor can enter goals during the ritual but cannot sign them off, so the person entering a goal and the person approving it are never the same when a monitor is involved.

<Warning>
Once a goal is both approved and marked complete in the End-of-Day Check, it is locked against casual re-marking in the ritual. Changing it after that point goes through the audited correction path instead.
</Warning>
