# Administrator Insights Guide

As an administrator, Zion&apos;s Insights feature helps you identify patterns and trends across your school that might otherwise go unnoticed. This guide covers how to use insights effectively for data-driven decision making.

## Your Insights Responsibilities

### Weekly (10 minutes)

- Review top insights on dashboard
- Investigate high-relevance patterns
- Dismiss addressed insights

### Monthly (20 minutes)

- Analyze insight patterns over time
- Generate fresh insights manually
- Share findings with leadership

---

## Accessing Insights

### Dashboard Widget

The administrator dashboard shows top 3 insights:

1. Log into Zion
2. Navigate to **Dashboard**
3. See **Latest Insights** widget
4. Click any insight for details
5. Click **View All** for complete list

### Full Insights Page

1. Navigate to **Analytics → Insights**
2. View all active insights
3. Filter and sort as needed

---

## Understanding the Dashboard

### Summary Cards

| Card | Description |
|------|-------------|
| **Total Active** | All current insights |
| **High Confidence** | Statistically reliable |
| **Medium Confidence** | Moderate reliability |
| **Avg Relevance** | Mean relevance score |

### Insight Cards

Each insight shows:

- **Type Icon:** Visual category indicator
- **Title:** Brief finding description
- **Description:** Detailed explanation
- **Confidence Meter:** Visual reliability
- **Relevance Score:** 0-100 importance
- **Created Date:** When generated

## Insight Types

### Day-of-Week Patterns

**What to Look For:**
- Days with significantly lower performance
- Consistent weekly patterns

**Example Actions:**
- Adjust Monday goals lower
- Plan engaging Friday activities
- Investigate specific day challenges

### Subject Correlations

**What to Look For:**
- Subjects that struggle together
- Shared learning challenges

**Example Actions:**
- Identify prerequisite gaps
- Create paired tutoring programs
- Review curriculum sequencing

### Centre Performance

**What to Look For:**
- Centres above or below average
- Outlier performance patterns

**Example Actions:**
- Support struggling centres
- Document high-performer practices
- Reallocate resources

### Seasonal Trends

**What to Look For:**
- Term-long performance changes
- Fatigue or momentum patterns

**Example Actions:**
- Address mid-term fatigue
- Celebrate term-end momentum
- Adjust term workloads

### Student Velocity

**What to Look For:**
- Students accelerating or declining
- Momentum shifts

**Example Actions:**
- Celebrate improving students
- Intervene with declining students
- Identify what&apos;s driving changes

### Subject Difficulty

**What to Look For:**
- PACEs with low completion rates
- Content that consistently challenges

**Example Actions:**
- Provide additional resources
- Adjust teaching approaches
- Consider prerequisite review

## Working with Insights

### Viewing Details

1. Click **View Details** on any insight
2. Side drawer opens with:
   - Full description
   - Affected students list
   - Student metrics
   - Export option

### Analyzing Affected Students

The detail view shows students involved:

| Column | Description |
|--------|-------------|
| **Name** | Student name |
| **Achievement Rate** | Their completion % |
| **Goals Met** | Completed goals |
| **Total Goals** | Goals attempted |

Use this to:
- Identify students for intervention
- Prioritize by severity
- Plan targeted support

### Dismissing Insights

When you&apos;ve addressed an insight:

1. Click **Dismiss** on the card
2. Insight is hidden from view
3. Won&apos;t reappear until new data

<Warning>
**Permanent Dismissal:** Dismissed insights don&apos;t return. Only dismiss after taking action or determining it&apos;s not relevant.
</Warning>

## Generating Insights

### Automatic Generation

Insights generate automatically:
- Weekly refresh cycle
- Old insights replaced
- Fresh analysis each time

### Manual Generation

Trigger immediate analysis:

1. Click **Generate Insights** button
2. System analyzes current data
3. New insights appear in minutes
4. Old non-dismissed insights replaced

**When to Generate Manually:**
- After significant data changes
- Before important meetings
- When investigating specific issues

## Filtering and Sorting

### By Confidence

Filter to reliability level:
- **All:** Show everything
- **High:** Most reliable
- **Medium:** Moderate reliability
- **Low:** Interpret with caution

### By Type

Focus on specific patterns:
- Day-of-Week
- Subject Correlation
- Centre Performance
- Seasonal Trend
- Student Velocity
- Subject Difficulty

### Default Sort

Insights sort by relevance score (highest first).

## Confidence Levels

### Understanding Confidence

| Level | Meaning | Sample Size |
|-------|---------|-------------|
| **High** | Very reliable | Large samples |
| **Medium** | Reliable | Moderate samples |
| **Low** | Use caution | Small samples |

### Confidence Meter

- **Green (100%):** High confidence
- **Blue (66%):** Medium confidence
- **Gray (33%):** Low confidence

### Acting on Low Confidence

Low confidence doesn&apos;t mean wrong:
- Consider the finding carefully
- Look for supporting evidence
- Don&apos;t dismiss automatically
- May become high confidence with more data

## Best Practices

### Regular Review

- Check insights weekly at minimum
- Don&apos;t let insights accumulate
- Prioritize by relevance score

### Prioritization

1. **High relevance + High confidence:** Act immediately
2. **High relevance + Medium confidence:** Investigate more
3. **Medium relevance:** Address when time permits
4. **Low relevance:** Consider dismissing

### Team Communication

- Share relevant insights in staff meetings
- Assign action items to appropriate staff
- Follow up on intervention results
- Celebrate pattern improvements

### Building Insight History

- Insights improve with more data
- New schools see limited insights initially
- Full intelligence emerges after 4-6 weeks
- Consistent data entry is key

## Troubleshooting

### No Insights Appearing

**Possible Causes:**
- Not enough data yet
- All insights dismissed
- No patterns detected

**Solution:**
- Wait for more data accumulation
- Check dismissed insights
- Generate manually

### Insight Seems Wrong

**Steps:**
1. Check confidence level
2. Verify underlying data
3. Consider sample size
4. May be valid but unexpected

### Cannot Dismiss

**Check:**
- You have administrator access
- Not a system-protected insight
- Try refreshing the page
