# Analytics Dashboard

Zion&apos;s Analytics Dashboard provides comprehensive data visualization across attendance, goal achievement, grading, and student performance. This guide covers all analytics capabilities.

## Accessing Analytics

1. Navigate to **Analytics** in the sidebar
2. Choose from available sections:
   - **Trends:** Time-series analysis
   - **Subjects:** Subject performance breakdown
   - **Centres:** Learning centre comparison
   - **Insights:** AI-generated patterns

---

## Analytics Hub

### Overview Page

The main analytics page provides quick navigation:

**Available Analytics:**
- Trend analysis over time
- Subject performance breakdown
- Centre comparisons
- AI-powered insights

**Student Analytics:**
- Grid of students with recent goal data
- Click any student to view their trend analysis
- Filter by centre or grade level

---

## Trends Analytics

### Time Series Chart

**What It Shows:**
Goal completion rates plotted over time

**X-Axis:** Dates in selected range
**Y-Axis:** Completion percentage (0-100%)

### Date Range Options

| Range | Description |
|-------|-------------|
| **7 Days** | Last week&apos;s data |
| **30 Days** | Last month&apos;s data |
| **Current Term** | From term start |
| **Custom** | Any date range |

### Trend Indicators

- **Trend Direction:** Improving / Stable / Declining
- **Average Rate:** Mean completion for period
- **Best Day:** Highest performing day
- **Worst Day:** Lowest performing day

### Interactive Features

- Hover over data points for details
- Click data point to see student breakdown
- Export chart data to CSV

---

## Subject Analytics

### Subject Performance Chart

**Visualization:** Horizontal bar chart showing performance by subject

**Metrics Per Subject:**
- Average completion rate
- Total goals in period
- Pass rate percentage
- Trend direction

### Subject Radar Overview

**Visualization:** Radar/spider chart for multi-dimensional view

**Shows:**
- Relative performance across all subjects
- Visual balance of subject strengths/weaknesses
- Quick identification of outliers

### Subject Details

Click any subject to see:
- Student breakdown for that subject
- PACE-level performance
- Historical trends
- Comparison to school average

---

## Centre Analytics

### Centre Comparison Chart

**Visualization:** Bar chart comparing learning centres

**Metrics Per Centre:**
- Completion rate
- Student count
- Goals completed
- Trend direction

### Outlier Highlighting

Centres are color-coded:
- **Green:** &gt;1 standard deviation above mean
- **Normal:** Within 1 standard deviation
- **Red:** &gt;1 standard deviation below mean

### Centre Metrics Table

Sortable table showing:

| Column | Description |
|--------|-------------|
| Centre Name | Learning centre identifier |
| Students | Active student count |
| Completion Rate | Average achievement |
| Goals This Week | Total goals set |
| Trend | Direction indicator |

### Centre Alerts

Banner highlighting:
- Centres needing immediate attention
- Centres with significant changes
- Resource allocation recommendations

## Real-Time Updates

### Auto-Refresh

**Dashboard Refresh:**
- Automatic refresh every 30 seconds
- Shows live data as goals are marked
- No manual refresh needed

**Focus Refresh:**
- Refreshes when you return to tab
- Ensures current data on view

### Cache Strategy

- Dashboards: 30-second cache (real-time feel)
- Analytics: 5-minute cache (stability)
- Insights: On-demand refresh

---

## Export Capabilities

### CSV Export

From any analytics view:
1. Click **Export** button
2. Select date range and filters
3. Download CSV file

**CSV Contains:**
- Raw data points
- Calculated metrics
- Timestamps
- All visible columns

### Chart Export

Export visualizations:
1. Click chart export icon
2. Choose format (PNG/SVG)
3. Download image file

## Best Practices

### Regular Review Schedule

| Frequency | Focus |
|-----------|-------|
| Daily | Dashboard snapshot |
| Weekly | Trends and patterns |
| Monthly | Centre comparisons |
| Quarterly | Subject analysis |

### Using Data Effectively

1. **Identify Patterns:** Look for recurring trends
2. **Compare Groups:** Centre vs centre, subject vs subject
3. **Drill Down:** Don&apos;t stop at high-level; investigate details
4. **Take Action:** Use data to drive interventions
5. **Monitor Results:** Track if interventions work

### Common Analysis Questions

- Which days have lowest performance?
- Which centres need support?
- What subjects are struggling?
- Are we improving over the term?
- Which students need attention?
