Goal Check entry is where an administrator, supervisor, or monitor sets the page range a student should complete in each assigned PACE. Goals are set on a weekly grid, one row per weekday and one column group per subject, and the same grid is used later to mark actual progress.

<Frame caption="The weekly goal setting grid for entering a student's page range per PACE.">
  <img src="/images/features/goal-check/setting-goals-1.png" alt="Weekly goal setting grid with start page and end page fields" />
</Frame>

## Who can set goals

Administrators, supervisors, and monitors can set goals. Supervisors and monitors only see students in the learning centres they are assigned to. Administrators see every centre.

## Opening the entry screen

<Steps>
  <Step title="Open Goal Check">
    In the sidebar, open Goal Check, then click Set Goals. This opens the Weekly Goal Setting screen at /goal-check/entry.
  </Step>
  <Step title="Choose a learning centre">
    Use the Learning Centre filter to scope the student list, or leave it on All Centres.
  </Step>
  <Step title="Choose a student">
    Use the Student selector, or the previous and next student buttons, or the left and right arrow keys, to move between students. A dot next to a name marks a student who has no goal set for the displayed week; a checkmark marks one who does.
  </Step>
</Steps>

<Tip>
Turn on &quot;Show only students without goals&quot; to filter the student list down to coverage gaps, or click &quot;Next without goal&quot; to jump straight to the next student missing a goal for the week.
</Tip>

## Entering a goal

Each student&apos;s grid shows one column per assigned PACE and one row per weekday (Monday through Friday). Click an empty cell to enter a goal.

**Fields:**

| Field | Behavior |
|---|---|
| Start page | Page number the student begins from that day. Type a value; there is no default. |
| End page | Page number the student should reach that day. Bounded by the PACE&apos;s total page count; the field rejects a value above it. |
| Total pages | End minus start plus one. Calculated and shown after the goal is set; not directly editable. |

Click Save to write the goal, or Cancel to discard the cell.

<Warning>
A PACE with no total page count recorded cannot be goaled. Its end-page limit resolves to zero, so no valid range can be entered. Ask an administrator to set the PACE&apos;s page count before setting a goal against it.
</Warning>

### Quick-template chips

While a cell is open, three chips (5, 8, 10) fill in a page range starting from wherever the student&apos;s PACE currently stands, clamped to the PACE&apos;s total page count. Confirm or adjust the prefilled values, then click Save. If the PACE is already complete, the chips have nothing to fill and a message says so.

### Deleting a goal

A goal that has not yet been marked can be deleted directly from its cell (no reason required, since nothing was recorded against it). A goal that has already been marked can only be removed through the audited correction path, available to administrators and supervisors.

## Bulk entry

Switch from the Single student tab to Bulk entry to set goals for several students in one pass instead of opening each student&apos;s grid individually.

<Steps>
  <Step title="Select students">
    Check the students in the centre roster you want to set goals for, or check the header box to select everyone.
  </Step>
  <Step title="Pick a page-count chip">
    Choose 5, 8, or 10 pages. Each selected student&apos;s goal continues from their current progress in every assigned PACE, clamped to that PACE&apos;s total pages.
  </Step>
  <Step title="Apply, or copy yesterday's goals">
    Click &quot;Apply [N]-page goals to selected&quot; to create the goals, or &quot;Copy yesterday&apos;s goals&quot; to copy the previous day&apos;s ranges onto the selected students instead.
  </Step>
</Steps>

The roster table shows each selected student&apos;s PACE count and whether they already have goals set, so gaps are visible before you apply anything.

<Note>
The full-screen morning and end-of-day ritual is a separate, guided way to run Goal Check for an entire centre in one session. See [The daily ritual](/features/goal-check/rituals) for how its goal-setting step differs, including its automatically derived start page.
</Note>
