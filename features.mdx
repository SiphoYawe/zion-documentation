Zion is a school-management app built for ACE (Accelerated Christian Education) schools. It covers daily Goal Check recording, attendance, PACE grading and moderation, behaviour and homework tracking, library and inventory management, meetings, calendar, communication, report cards, admissions, analytics, and billing, alongside a parent portal and a student portal.

Zion has five roles: administrator, supervisor, monitor, parent, and student. Each feature area below states which roles use it. See <a href="/reference/roles-permissions">Roles and permissions</a> for the full permission matrix.
