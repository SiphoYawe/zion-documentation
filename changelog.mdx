# Changelog

Track the features, improvements, and fixes shipped in Zion.

## Version 1.0.0: Launch (2025-11-22)

### Initial Release

Zion Phase 1 (Micro-MVP) launched with Epic 1 and Epic 2 complete.

### Epic 1: Foundation &amp; School Setup

**School Administration:**
- Multi-tenant school organization setup
- School profile configuration with logo upload
- Academic term management (create, edit, activate terms)
- Learning centre creation and management
- Timezone configuration for accurate date and time handling

**User Management:**
- Role-based access control (Administrator, Supervisor, Monitor, Student, Parent)
- Staff invitation system via email
- Centre-based user assignments
- User activation and deactivation

**Student Roster:**
- Individual student profile creation
- CSV bulk import for roster setup
- Student assignment to learning centres
- Parent contact information

**PACE Curriculum:**
- Complete ACE PACE catalog (144 PACEs, 1001 to 1144)
- 12 subjects across 12 levels
- Individualized PACE assignments per student
- Search and filter across the PACE catalog

### Epic 2: Goal Check Intelligence

**Daily Goal Setting:**
- Quick goal entry interface for all students
- Auto-calculated goal total pages
- Copy previous day&apos;s goals
- Auto-save
- Tablet-optimized touch interface

**Progress Tracking:**
- End-of-day progress marking
- Actual vs. goal variance calculation
- Optional notes for struggling students
- Color-coded completion status
- Real-time dashboard updates

**Dashboards:**
- **Administrator Dashboard:**
  - Today&apos;s school-wide snapshot
  - Weekly trends visualization
  - Centre performance comparison
  - At-risk student alerts
  - Quick action cards
- **Supervisor Dashboard:**
  - Centre-scoped data view
  - Student progress table
  - Trends for the supervisor&apos;s centres
  - Centre-specific alerts
  - Report readiness indicators

**Automated Reporting:**
- Daily centre reports (PDF and CSV)
- Weekly school reports (PDF and CSV)
- Centre summary statistics
- Student-by-student breakdowns
- At-risk student identification
- Subject performance analysis
- Trend indicators (current vs. previous week)

**Data Export:**
- CSV export with flexible filters
- Full audit trail included in exports
- Background processing for large exports
- Export history tracking
- Filters by date range, centre, student, status, and subject

**Mobile Optimization:**
- Responsive design for tablets
- Touch-friendly interface (&gt;=44px touch targets)
- Native mobile inputs (number pads, date pickers)
- Offline detection with status banner
- Load times under 3 seconds

### Technical Infrastructure

- Next.js 15 with App Router
- TypeScript with strict mode
- Supabase PostgreSQL database
- Clerk authentication and organizations
- Tremor UI component library
- Hugeicons icon library
- Row Level Security (RLS) for multi-tenancy
- Automated database migrations
- Vercel deployment

### Documentation

- Documentation site at docs.zionapp.co
- Quickstart guide
- Administrator onboarding handbook
- Supervisor daily workflow guide
- Feature documentation
- FAQ and support resources
