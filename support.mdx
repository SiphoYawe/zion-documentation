## Before You Contact Support

Help us help you faster by providing:

1. **Your School Name:** So we can locate your account
2. **User Role:** Are you an Administrator, Supervisor, or Monitor?
3. **What You Were Trying to Do:** Describe the task you were attempting
4. **What Happened:** Include any error messages you saw
5. **Screenshots:** If possible, attach screenshots showing the issue
6. **Browser & Device:** E.g., "Safari on iPad" or "Chrome on Windows"

## Common Issues

### Account & Login

**Can&apos;t log in:**
1. Verify email address is correct
2. Try password reset
3. Check email for verification link (if new account)
4. Contact support if still unable to log in

**Didn&apos;t receive invitation email:**
1. Check spam/junk folder
2. Verify email address with administrator
3. Request administrator to resend invitation
4. Contact support if still not received

### Goal Check

**Students not appearing in Goal Check:**
- Ensure students have PACE assignments
- Verify students are active (not deactivated)
- Check that you&apos;re viewing the correct date

**Can&apos;t generate report:**
- Verify all students in centre are marked
- Check readiness indicator shows "Ready"
- Try refreshing the page
- Contact support if issue persists

### Data Import

**CSV import failing:**
1. Download template and compare format
2. Check for special characters in names
3. Verify centre names match exactly
4. Ensure dates are in YYYY-MM-DD format
5. Remove empty rows at bottom
